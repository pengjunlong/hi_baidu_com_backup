gdb小结

backtrace   各级函数调用

finish 运行到当前函数结束

frame 1  选择栈帧

info locals 当前栈帧局部变量

list 列出源代码

next 执行下一个语句

print 打印表达式的值

quit 退出

set var 修改变量的值

start 开始执行程序，停在main第一行语句前

step 执行下一行，可进入函数调用

 

break 1 设置断点

break main 在函数前设置断点

break 6 if i==1    设置条件断点

continue 连续执行直到断点

delete breakpoints 1     删除断点

display i       跟踪查看变量

disable breakpoints 1     禁用断点

enable 1      启用断点

info breakpoints    查看断点

run   从头连续运行

undisplay  i       取消跟踪查看

 

watch   input[1]          设置观察点

info watchpoints    查看当前设置了哪些观察点

x/6b   input        从input开始打印6个字节的存储单元的内容
