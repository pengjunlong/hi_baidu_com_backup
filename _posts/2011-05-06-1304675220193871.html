Description
还记得大明湖畔的女生节吗？在宏福一年一度的女生节上，女生们可以把自己的愿望写在纸条上，等待男生帮自己实现，多么浪漫的活动。

作为一个男生，丁丁自然免不了好奇的想去看看。在一片大果粒的渴求中，丁丁发现了一份奇怪的与众不同的纸条，上面写着：

“在学二旁边的松林，有几棵松树环绕成一圈，从标有红色丝条那棵数起，每隔7棵树就将一棵树标为蓝色（接下来数树的时候就不用在考虑这棵了），最后剩下的那棵树下埋着我的正确的号码（注意其他树下埋的号码都是错误的哦）。你有本事给我发短信吗？期待ing～”
丁丁很是好奇，这是怎么样的一只女生呢？他决心找出那个号码，但又不知道号码到底埋在哪棵树下。于是，丁丁暴力的把那儿挖了个遍，却悲剧的发现了一堆不相同的号码，明显的是，其中只有一个号码是对的。请你帮助丁丁判断到底哪个号码才属于那个神秘的女生呢。
Input
多组数据输入，每组数据占一行，一组数据只有一个数字n，表示那儿有n棵树（2<=n<=50000）

Output
对每组数据输出一行，该行只输出一个数，即埋有正确的号码的那棵树的编号。


SampleInput

3
368
SampleOutput
3

337


hint：我们假定第一棵树的编号是1，接下来的数的树的编号是2，etc；假如这里有3棵树，则从第一棵开始数起的话，第一次数到第一棵（1，2，3，1，2，3，1），将其排除出去，第二次数到第二棵，将其排除出去，最后剩下的就是第三棵。
